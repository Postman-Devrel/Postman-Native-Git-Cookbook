# SdkGenerator TypeScript SDK 1.0.0

Welcome to the SdkGenerator SDK documentation. This guide will help you get started with integrating and using the SdkGenerator SDK in your project.

This SDK was generated by [Postman](https://www.postman.com)

## Versions

- SDK version: `1.0.0`

## About the API

REST API for Intergalactic Bank - Managing accounts and transactions.

## Authentication

Most endpoints require an API key passed in the `x-api-key` header.
Use the `/api/v1/auth` endpoint to generate a new API key.

## Available Currencies

- COSMIC_COINS
- GALAXY_GOLD
- MOON_BUCKS

## Account Types

- STANDARD
- PREMIUM
- BUSINESS

# Setup & Configuration

## Supported Language Versions

This SDK is compatible with the following versions: `TypeScript >= 4.8.4`

## Installation

To get started with the SDK, we recommend installing using `npm` or `yarn`:

```bash
npm install sdk-generator
```

or

```bash
yarn add sdk-generator
```

## Setting a Custom Timeout

You can set a custom timeout for the SDK's HTTP requests as follows:

```ts
const sdkGenerator = new SdkGenerator({ timeout: 10000 });
```

# Sample Usage

Below is a comprehensive example demonstrating how to authenticate and call a simple endpoint:

```ts
import { SdkGenerator } from 'sdk-generator';

(async () => {
  const sdkGenerator = new SdkGenerator({});

  const { data } = await sdkGenerator.accounts.listAccounts();

  console.log(data);
})();
```
